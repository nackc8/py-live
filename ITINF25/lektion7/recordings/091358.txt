       pip is a Python package installer, recommended for
       installing Python packages which are not available
       in the Debian archive.  It can work  with  version
       control  repositories  (currently only Git, Mercu‐
       rial, and Bazaar repositories), logs output exten‐
       sively, and prevents partial installs by download‐
       ing all requirements before starting installation.

       command to use when installing packages for Python
       3.
