# Separat miljö rekommenderas!

Använd modulen [venv][1]:

Skapa en ny miljö i katalogen .venv:

```bash
python -m venv .venv
```

Aktivera miljön för att installera paket i den:

```bash
source .venv/bin/activate
```

[1]: https://docs.python.org/3.11/library/venv.html
